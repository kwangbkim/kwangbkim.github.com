Octopress: Static Blogging

13 October 2012

I’ve been running this blog for a few months now without giving proper credit to Octopress. If you haven’t heard of it, Octopress is a website generator primarily for blogging, on top of the ruby jekyll framework. It’s easy to use (little to no Ruby experience required), is free and hooks into Github Hosting (also free) quite nicely. If you’re thinking about starting a site it’s worth checking out.
